# Vizsga Projekt

Ez a repozitórium tartalmazza a **vizsga projekthez** kapcsolódó fájlokat, dokumentumokat és forráskódokat. A cél egy egyszerű CRUD alapú webalkalmazás létrehozása, amely termékeket és hirdetéseket kezel, felhasználói véleményekkel és képfeltöltési lehetőséggel.

## 📁 Tartalom

- `videogshop.sql` – Az adatbázis szerkezete és inicializáló SQL parancsok
- `videoghoposcuc` – Backend CRUD logika (javított értékkezelés)
- `MainGSHOP.zip` / `VideoGshop.zip` – Teljes projekt archívum (frontend + backend)
- `VideoGSHOP.drawio.png` – Adatmodell diagram (Draw.io)
- `projektleírás.docx` – Részletes projektleírás és követelmények
- `README.md` – Ez a fájl
- `teszt` – Tesztelési célú fájlok vagy mappák

## 🧩 Funkciók

- Felhasználók regisztrációja és jogosultságkezelése
- Termékek hozzáadása, módosítása, törlése
- Hirdetések létrehozása termékekhez
- Vélemények írása és értékelés
- Képek feltöltése hirdetésekhez

## 🛠 Technológiák

- **Backend**: Node.js + Express + Prisma ORM
- **Adatbázis**: MySQL
- **Frontend**: HTML/CSS/JS (archívumban)
- **Diagram**: Draw.io
